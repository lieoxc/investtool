# InvesTool Frontend

基于 React + TypeScript 的前端应用

## 技术栈

- React 18
- TypeScript
- Ant Design (UI组件库)
- React Router (路由)
- Axios (HTTP客户端)
- ECharts (图表库)

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm run build
```

## 项目结构

```
src/
├── components/     # 通用组件
├── pages/         # 页面组件
├── services/      # API服务
├── types/         # TypeScript类型定义
├── utils/         # 工具函数
└── App.tsx        # 应用入口
```
